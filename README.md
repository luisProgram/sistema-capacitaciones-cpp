# sistema-capacitaciones-cpp
Sistema en C++ para evaluar promedios de trabajadores usando matrices y validaciones.
# Sistema de Capacitaciones CAXTECH

Este es un proyecto desarrollado en **C++** como parte de mi curso Fundamentos en programación (1er ciclo). Es un sistema de consola diseñado para gestionar y evaluar las calificaciones de los empleados en distintos módulos de capacitación.

## Características
* **Validación robusta:** Uso de bucles `do-while` para evitar el ingreso de datos erróneos (números negativos, caracteres inválidos).
* **Gestión de memoria:** Limpieza del buffer de entrada usando `cin.ignore()` para combinar correctamente ingresos numéricos y cadenas de texto (`getline`).
* **Algoritmo de búsqueda:** Implementación de la lógica "Rey de la Colina" para determinar automáticamente al empleado con el mejor promedio final.
* **Interfaz amigable:** Uso de menús claros y limpieza de pantalla (`system("cls")`).

## Tecnologías
* Lenguaje: C++
* Librerías: `<iostream>`, `<string>`, `<stdlib.h>`

## Lo que aprendí
Con este proyecto consolidé mis conocimientos en estructuras de control (for, while, do-while, if-else), validación de datos (programación defensiva) y manejo básico de la terminal.
